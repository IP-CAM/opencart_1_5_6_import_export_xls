Freeware module "As is..." sharing
Please do not remove authors contacts from the module interface

The module is for import|export products and categories data to DB under Opencart.
The module was tested under Opencart 1.5.6.4, can be modified for other Opencart versions of 1.5.x seria

Modules features:
 - it can add new products and categories without removing data uploaded before
 - it can update part of data without deleting data from DB
 - it can update categories split from products
 - it can export data to Excel split

The module reduce CPU loads.

Requirments
1. Opencart 1.5.6.х (tested under version 1.5.6.4) 
2. VQMOD 2.5.1 (tested)

Setup
1. Copy folders and files from "Upload" to main folder of your site
2. Turn on access to the module for user group
3. Open module in menu "System > Import / Export"

Распространяется свободно по принципу "как есть".
При использовании как есть прошу оставлять контактные данные автора в интерфейсе модуля.

Необходимые доработки осуществляются приобретателем самостоятельно, либо на коммерческой основе.

Модуль для загрузки/выгрузки товаров и категорий в базу Opencart из файла Excel.
Представляет собой переработанный свободно распространяемый модуль автора JNeuhoff

Преимущества модуля перед базовым:
 - добавлена возможность добавлять товары и категории без удаления ранее созданных;
 - добавлена возможность частичного обновления товаров и категорий, без удаления остальных позиций;
 - добавлена возможность раздельной загрузки категорий; 
 - добавлена выгрузка в отдельные файлы категорий и товаров;
 - переработаны запросы к базе данных.
 
Модуль существенно уменьшает нагрузку на систему в случаях когда требуется частичное обновление (особенно актуально для магазинов с большим ассортиментом товаров).
 
Структура файла Excel выполнена под конкретный проект (возможно потребует доработки приобретателем).
 
Требования к системе:
1. Opencart версии 1.5.6.х (протестировано под версией 1.5.6.4), установка под другие версии осуществляется приобретателем по собственной инициативе
2. VQMOD версии 2.5.1 (протестировано), установка под другие версии осуществляется приобретателем по собственной инициативе

Порядок установки:
1. Скопируйте  файлы и папки из папки upload в корень сайта;
2. Установите право администратора на пользование модулем
3. Переход к модулю осуществляется через меню СИСТЕМА > Экспорт / Импорт
